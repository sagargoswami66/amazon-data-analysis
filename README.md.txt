# Amazon Data Analysis Project

This project showcases an end-to-end data analysis workflow using an Amazon dataset sourced from Kaggle.

## Project Objective
To clean, analyze, and visualize Amazon product data in order to extract meaningful business insights.

## Project Workflow

### 1. Data Cleaning & Exploratory Data Analysis (Python)
- Cleaned and preprocessed the dataset
- Handled missing values, duplicates, and data formatting
- Performed:
  - Univariate analysis
  - Bivariate analysis
  - Multivariate analysis
- Created visualizations to identify trends and relationships

### 2. SQL Analysis
- Wrote 8 SQL queries to extract key insights from the data
- Analysis included:
  - Top-selling products
  - Category-wise performance
  - Sales trends over time
- SQL scripts are available in the `SQL` folder

### 3. Power BI Dashboard
- Designed an interactive Power BI dashboard
- Visualized key metrics and insights using charts and KPIs
- Dashboard file and screenshot are included in the `PowerBI` folder

## Tools & Technologies Used
- Python (Pandas, Matplotlib, Seaborn)
- SQL
- Power BI
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization

## Dataset
The original dataset is not included in this repository.
It was sourced from Kaggle and used for analysis purposes only.

## Key Learnings
- Hands-on experience with real-world data cleaning
- Strengthened SQL querying and analytical thinking
- Improved data storytelling using Power BI dashboards
